User Guide

Getting Started

  1. Create an account or log in
  2. Review SaaS Terms of Use Agreement in Dashboard
  3. Connect your hosting server
  4. Choose website category
  5. Select a theme
  6. Generate Top Menu
  7. Generate content
  8. SEO and Performance Optimization
  9. Launch your website

Server Requirements

  • cPanel access
  • PHP 7.4 or higher
  • MySQL 5.6 or higher
  • Apache or Nginx web server

Features

  • Automated WordPress installation
  • Theme customization
  • Content generation
  • Image generation Description
  • SEO optimization
  • Performance optimization
